  2 Create UEFI win10 installation USB key in Linux
  3 
  4 { steps }
  5 
  6 1. Download ISO
  7 
  8 2. Create GPT and 2 partitions on USB key(fat32 & NTFS).
  9 
 10 3. Copy files to USB key.
 11 
 12 4. Insert USB key to new PC, select UEFI boot.
 13 
 14 { overview }
 15 
 16 The Windows installation will start on the FAT32 partition, but when encountering the empty "sources" folder it will look for the "sources/ei.cfg" file on the NTFS partition and continue the installation without any prompts.
 17 
 18 NOTE
 19 
 20 "dd" does NOT work for win10 1909.
 21 
 22 "UNetbootin" does NOT work either.
 23 
 24 "WinUSB" does NOT work with Windows 10.
 25 
 26 some PC UEFI boot can NOT find ntfs partition.
 27 
 28 Install.wim is larger than 4G, so we can NOT copy it into a fat32 partition.
 29

 47 { Create working directories }
 48 
 49 fat32           to mount USB key's fat32 partition.
 50 ntfs            to mount USB key's ntfs partition.
 51 iso             to mount win10 iso.
 52 $ sudo mkdir -p /mnt/win10/{fat32,ntfs,iso}
 53 
 54 $ sudo mount ~/Desktop/Win10_1909_English_x64.iso /mnt/win10/iso
 55 
 56 { umount USB }
 57 
 58 Some Linux distributions will auto mount USB key.
 59 
 60 Check USB key
 61 $ mount
 62 
 63 Umount all the parts of USB. e.g.
 64 $ umount /media/abc/66F0-46A2
 65 
 66 { create parts }
 67 
 68 Check USB key device name
 69 $ lsblk
 70 
 71 Delete all the partitions
 72 // /dev/sdb     replace it with real device name
 73 $ sudo sfdisk --delete /dev/sdb
 74 
 75 Show partition table
 76 $ sudo parted /dev/sdb print
 77 
 78 Create gpt table
 79 $ sudo parted -s /dev/sdb mklabel gpt
 80 
 81 Create FAT32 part
 82 $ sudo parted -s /dev/sdb mkpart primary fat32 0% 1024M
 83 $ lsblk -f
 84 $ sudo mkfs.fat -F32 /dev/sdb1
 85 
 86 Create NTFS part
 87 $ sudo parted -s /dev/sdb mkpart primary ntfs 1024M 100%
 88 $ mkfs.ntfs -Q -v /dev/sdb2 -s 4096
 89 
 90 Show partitions on sdb
 91 $ sudo parted /dev/sdb print
 92 
 93 { mount USB key parts }
 94 
 95 sdb1    part of fat32, verify by lsblk.
 96 $ sudo mount /dev/sdb1 /mnt/win10/fat32/
 97 $ sudo mount /dev/sdb2 /mnt/win10/ntfs/
 98 
 99 { copy files to fat32 part }
100 
101 Copy all files of iso, except "sources" dir to fat32.
102 $ sudo rsync -avr --exclude 'sources' /mnt/win10/iso/ /mnt/win10/fat32/
103 
104 Create a empty "sources" dir
105 $ sudo mkdir /mnt/win10/fat32/sources
106 
107 $ vi ei.cfg
108 /*{{{*/
109 [CHANNEL]
110 Retail
111 /*}}}*/
112 
113 $ sudo cp ei.cfg /mnt/win10/fat32/sources/
114 
115 $ sudo cp /mnt/win10/iso/sources/boot.wim /mnt/win10/fat32/sources/
116 
117 { copy files to NTFS part }
118 
119 $ sudo rsync -avr /mnt/win10/iso/ /mnt/win10/ntfs/
120 
121 { umount 2 parts }
122 
123 $ sudo umount /mnt/win10/fat32/
124 $ sudo umount /mnt/win10/ntfs/
125 
126 Disconnect USB key
127 
128 { Config PC to be installed }
129 
130 // BIOS Keys will be different.
131 
132 POST -> F1 (BIOS)
133 
134 UEFI/Legacy Boot:       Both
135 Boot Priority:          UEFI first
136 CSM Support:            Yes
137 
138 Save, exit.
139 
140 POST -> F12 (Boot menu) -> SELECT USB HDD
